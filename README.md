# Ordeer

### An online ordering platform for restaurants
### Created by Braden Shipley, W. Andrew Montgomery, and Eric Willroth

Live site: ordeer.net

# Technology Used

React | Node | Axios | Firebase database | Material-UI | Nodemailer | Jest
